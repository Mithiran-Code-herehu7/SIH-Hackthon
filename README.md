# SIH-smart_tourist_safety_system
The project i worked on during my first year of college (for a hackathon).
**Full-stack prototype** using AI, Blockchain, Geo-Fencing for tourist safety monitoring.

## Live Demo
[Deployed Link](https://mithiran-codes-here.github.io/SIH-smart_tourist_safety_system/)

- Features: Real-time tracking, Panic button, Digital Tourist ID
